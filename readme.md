## MyProyectoDevOps_Bautista

## Descripción del proyecto:

Este repositorio corresponde al proyecto DevOps de la Unidad 3, enfocado en el proceso de liberación, despliegue continuo y monitoreo de una aplicación web.

El proyecto consiste en el desarrollo de una plataforma web para la gestión de actividades, donde los usuarios podrán administrar proyectos, crear tareas, asignar responsables, cambiar estados y visualizar el avance del trabajo.

Este repositorio servirá como base para aplicar prácticas DevOps como control de versiones, integración continua, despliegue continuo, documentación y monitoreo.


## Objetivo del proyecto:

Desarrollar una plataforma web aplicando buenas prácticas DevOps, con el propósito de organizar el trabajo en equipo, automatizar procesos de desarrollo y preparar el proyecto para futuras etapas de despliegue y monitoreo.


## Tecnologías que se usaran:

Las tecnologías que se utilizarán o podrán integrarse durante el desarrollo del proyecto son:

- HTML
- CSS
- Bootstrap
- JavaScript
- Node.js
- Express
- Base de datos(mysql)
- Git y GitHub
- GitHub Actions
- Render o Railway
- Herramientas de monitoreo
- Loki
- Prometheus
- Giafana
